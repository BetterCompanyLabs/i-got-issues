I Got Issues is a  lightweight issue tracker for solo builders.
It is supposed to be fast and intentionally simple. No onboarding required, just start right away organizing your issues.

This one goes out to all my solo developers, indie makers, vibe-coders ...
-> Your TODO.md file has grown out of control?
-> You don't want to spend hundreds of bucks on enterprise tools?
-> you don't have time to learn about dozens of features / sub-menus etc. etc.? 

I Got Issues focuses on one thing: tracking issues quickly and efficiently.

FEATURES
* Easy creation of multiple projects
* Add and track issues with following attributes:
    * Title
    * Long Text
    * Status
    * Priority
    * Topic/Tag
    * Effort
    * Due date
    * Assignee
* Keyboard shortcuts for fast item workflows
* Multi-select and bulk actions
* Automatic topic allocation via #topic tags in titles
* JSON import/export for complete workspace backups
* Dark and light mode

ECONOMICS
Let's be realistic: this will not be the next Monday / Linear. I am not planning to get rich with this thing. 
I Got Issues is a mix of passion about organizing work and a little glimps of hope to earn a few extra bucks per month.

So...
* You can either subscribe for 0,99 EUR per month and use all the features including cloud storage of your issues
* or you run the FREE "local" version that stores your issues in your browser cache and you can use the Export/Import JSON feature to backup and update your issues


The application itself is simple:
* HTML, CSS, Vanilla JavaScript

I Got Issues is designed for people who just want to keep track of their work without spending more time managing a tool than building their project.

Because our TODO.md deserves better.
